This project contain some methods / algorithms such as:

- K-Means
- Oriented Fast and Rotated BRIEF (ORB)
- Prewitt Sobel
- Scale Invariant Feature Transform (SIFT)
- Convolutional Neural Network (CNN)
- Histogram of Oriented Gradients (HOG)
